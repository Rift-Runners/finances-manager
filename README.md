# Finances Manager

- Project with the objective of creating a responsive, clean and simple finances manager;
- Using Java, based on WebServices (RESTful) and persisted with JPA.

## Tools and Frameworks

- AngularJS
- Foundation
- JQuery
- Remodal
- Maven (Later switch to Gradle)
- Hibernate (DB: MySQL)
